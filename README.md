## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create a local env file for the contact form:
   `copy .env.example .env`
3. Fill in the `VITE_EMAILJS_SERVICE_ID`, `VITE_EMAILJS_TEMPLATE_ID`, and `VITE_EMAILJS_PUBLIC_KEY` values from EmailJS.
4. Run the app:
   `npm run dev`
